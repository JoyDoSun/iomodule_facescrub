This is a code to download and save images (in a specific manner described below) from facescrub dataset using the url provided.

The code downloads images in 4 different directories namely :

faces_training  n
faces_testing
non_faces_training
non_faces_testing

change n and m to change no of training and testing images respectively.

errors may occur because of non existing links or images modify the code accordingly for solving them.

it is recommended that you keep the no less and run the code multiple times to get large no of images if error persists.

INSTRUCTIONS:

Required:
Python 2.7
OpenCV
Numpy


copy Face Scrub (.txt) files for actors and actresses in the same directory.
run iomodule.py
